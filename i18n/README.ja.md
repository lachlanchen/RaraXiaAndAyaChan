`RaraXiaAndAyaChan` は Lachlan Chen による、かわいくて少しSFな物語アーカイブです。ララ侠、アヤちゃん、ササ君が、月面基地、彗星、ブラックホール、超新星、魔法の森、そして食べ物の勘違いをめぐって冒険します。

このリポジトリには、シリーズ設定、40話の宇宙冒険、15秒動画プロンプト、小云雀 / Seedance の制作メモ、ローカルツールが入っています。

おすすめ: `Lala-Aya-Sasa-draft/universe-adventure-40-episodes-v2/` と `references/prompts/`。
