# 宇宙农场大冒险

这是一个连续 40 集、每集约 15 秒的啦啦侠、阿芽酱、飒飒君宇宙冒险短片系列。故事围绕一张拼皮星图、会发光的宇宙种子、迷路外星人、彗星、飞船、宇宙农场和一场跨星系的搞笑战争展开。

## 文件结构

```text
series-bible.md
episodes-01-40.md
xiaoyunque-15s-prompts.md
```

## 默认生成设定

- 角色：啦啦侠、阿芽酱、飒飒君。
- 风格：温暖可爱、科幻冒险、轻松搞笑、电影级动画质感。
- 时长：每集 15 秒。
- 结构：每集有冲突、笑点、动作和结尾钩子。
- 连续性：上一集结尾钩子会推动下一集开场。
- 默认要求：不要字幕，不要生成任何字幕、说明文字、下三分之一文字或画面文字。

## 使用方式

优先从 `xiaoyunque-15s-prompts.md` 复制单集提示词。需要改成更长片时，可以参考 `episodes-01-40.md` 的完整连续剧情。
