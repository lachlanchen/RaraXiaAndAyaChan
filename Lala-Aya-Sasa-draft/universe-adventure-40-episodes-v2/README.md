# 宇宙农场大冒险 v2

这是 `universe-adventure-40-episodes` 的扩展版，旧版本保持不动。本版本加入三个重要配角：猪猪侠、牛魔王、恐龙，并把每集写成更适合 15 秒短片的完整版本：冲突明确、对话更丰富、笑点更密、结尾继续吊住下一集。

## 新增配角

- 猪猪侠：银河农场临时站长，负责菜单、物资、合同和奇怪但有效的计划。
- 牛魔王：星际牧场主，外表凶，内心很讲义气，前期误会三人，后期成为强力队友。
- 恐龙咚咚：从彗星蛋里孵出来的小恐龙，体型大、胆子小、力气惊人，经常把敌人吓跑后自己先道歉。

每集 15 秒，中文对白，温暖可爱、科幻冒险、轻松搞笑、电影级动画质感。对话要刚好覆盖 15 秒，不要太密，也不要空。不要字幕，不要生成任何字幕、说明文字、下三分之一文字或画面文字。
