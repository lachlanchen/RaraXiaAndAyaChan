# LALACHAN Series Registry

This folder is the canon layer for reusable story systems. Individual daily
drafts stay in `references/stories/`; reusable worlds, characters, educational
rules, and visual props live here.

## Active Series

### Red Sand Paper Kite

Path: `references/series/red-sand-paper-kite/`

Purpose: a magical red-sand adventure about paper kites, bronze faces, gold
birds, repair, shared roads, and ordinary kindness. It can lightly echo
VoidAbyss motifs while staying cute and short-video friendly.

### Daily Custom

Path: `references/series/daily-custom/`

Purpose: standalone funny daily videos. These can use any setting, but should
still keep the trio's character voices and recurring props stable.

### Mars + Japan + Sanxingdui

Path: `references/series/mars-japan-sanxingdui/`

Purpose: generate original stories by reading one page-like unit from a Mars
source, one from Japanese history, and one from Sanxingdui, then transforming
their motifs into a smooth imaginative LALACHAN scene. This is the preferred
method when the user asks for Mars + Japan + Sanxingdui continuation stories.

### Chip Factory Adventures

Path: `references/series/chip-factory-adventures/`

Purpose: teach chip manufacturing, AI accelerators, CPU/GPU/NPU ideas, Apple
silicon, NVIDIA-style AI factories, Foxconn-style manufacturing, MediaTek edge
AI, Arm, RISC-V, packaging, memory, and supply-chain concepts through funny
stories.

### Robot, Drone, And AR Lab

Path: `references/series/robot-drone-ar-glass/`

Purpose: teach robotics, drones, dexterous hands, simulation, perception,
control, embodied AI, and AR/spatial computing through physical adventure.

## Universal Prompt Rule

Every educational video should still be a story first:

1. A visual problem appears.
2. One character misunderstands it in a funny way.
3. The robot or word-card explains one concept physically.
4. Aya Chan makes the careful interpretation.
5. The concept helps solve the scene.
6. End with a cliffhanger or tiny joke.
