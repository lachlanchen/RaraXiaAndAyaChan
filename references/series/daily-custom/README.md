# Daily Custom Stories

This folder describes the non-canon daily prompt style. These stories can be
funny, seasonal, topical, or experimental, and they do not need to follow the
red-sand continuity.

## Existing Examples

- Earth-moon elevator and Mars rope travel.
- Tokushima robot company, robot rock-paper-scissors, and panda cat.
- Firefly cave after cicada rain.
- Typhoon, ping-pong, shark, and flying Sasa.
- Hokkaido dolphin and shark riding.
- Alps hamburg / hamburger wordplay.

## Rules

- Keep Lala Xia, Aya Chan, and Sasa Kun consistent.
- Use Chinese as the main language unless another language is part of the joke.
- Keep action clear enough for a 15-second or 1-minute video.
- Save story drafts in `references/stories/`.
- Save generation prompts in `references/prompts/`.
- If a daily story starts forming a larger continuity, promote it into a new
  folder under `references/series/`.

## Good Daily Story Shape

1. One strong visual premise.
2. One misunderstanding or pun.
3. One warm correction by Aya Chan.
4. One physical gag by Sasa Kun.
5. One final hook or punchline by Lala Xia.
